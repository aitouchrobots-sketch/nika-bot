# 🏔️ Ника — ИИ-бот экспедиции на Мачу-Пикчу

Telegram-бот с искусственным интеллектом для сопровождения участников экспедиции.

## Что умеет Ника
- 📅 Отвечает на вопросы о программе экспедиции по дням
- 👥 Знает всех участников (из CSV-файла)
- ☕ Подбирает собеседников для кофе-брейков по компетенциям
- 💬 Работает в групповом чате (реагирует на "Ника,")

---

## ШАГ 1 — Создай бота в Telegram

1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя: `Ника` (или любое)
4. Придумай username: например `nika_expedition_bot`
5. BotFather пришлёт **токен** — сохрани его!

---

## ШАГ 2 — Получи Anthropic API ключ

1. Зайди на [console.anthropic.com](https://console.anthropic.com)
2. Раздел **API Keys** → **Create Key**
3. Скопируй ключ — он начинается с `sk-ant-...`

> ⚠️ На ключ нужно положить деньги (минимум $5). Бот дешёвый — на всю экспедицию хватит $1-2.

---

## ШАГ 3 — Деплой на Railway (бесплатно, 5 минут)

### 3.1 Загрузи код на GitHub
1. Создай аккаунт на [github.com](https://github.com) если нет
2. Создай новый репозиторий (New Repository), назови `nika-bot`
3. Загрузи все файлы из этой папки в репозиторий

### 3.2 Задеплой на Railway
1. Зайди на [railway.app](https://railway.app)
2. **New Project → Deploy from GitHub repo**
3. Выбери репозиторий `nika-bot`
4. Railway сам найдёт `railway.toml` и запустит бота

### 3.3 Добавь переменные окружения
В Railway → твой проект → **Variables** добавь:
```
TELEGRAM_BOT_TOKEN = твой_токен_от_BotFather
ANTHROPIC_API_KEY  = твой_ключ_sk-ant-...
```

5. Нажми **Deploy** — бот запустится!

---

## ШАГ 4 — Загрузи список участников

1. Заполни файл `participants_example.csv` реальными данными участников
2. Переименуй в `participants.csv`
3. Отправь этот файл **прямо в Telegram чат боту** (или в группу)
4. Ника ответит: "✅ Список участников обновлён!"

### Формат CSV (столбцы):
| name | company | role | expertise | interests |
|------|---------|------|-----------|-----------|
| Иван Иванов | МояКомпания | CEO | IT, финансы | инвестиции, стартапы |

---

## ШАГ 5 — Добавь Нику в групповой чат

1. Открой групповой чат участников
2. **Добавить участника → найди @nika_expedition_bot**
3. Сделай бота **администратором** (иначе не сможет читать сообщения)
4. Обращайтесь: **"Ника, куда едем завтра?"**

## Как пользоваться

### Вопросы о программе:
```
Ника, что у нас завтра?
Ника, программа на 13 апреля?
Ника, когда вылет?
Ника, какой отель в Куско?
```

### Кофе-брейк / знакомство:
```
Ника, хочу познакомиться с кем-нибудь
→ Ника спросит твоё имя

Иван Петров
→ Ника найдёт тебя в списке и предложит 2 человека для разговора
  с объяснением, почему вы совпадаете по темам
```

## Альтернатива Railway — запуск на своём компьютере

```bash
# 1. Установи Python 3.11+
# 2. Установи зависимости:
pip install -r requirements.txt

# 3. Создай файл .env (скопируй из .env.example):
cp .env.example .env
# Отредактируй .env — вставь токены

# 4. Запусти:
python bot.py
```

> Компьютер должен быть включён пока работает бот.
> Для постоянной работы — Railway или VPS.
